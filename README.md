Controller Music
automatic Spotify volume ducking for TrackAudio transmissions
Controller Music is a lightweight Windows utility that automatically lowers your Spotify volume whenever TrackAudio detects a radio transmission — whether it’s you transmitting (TX) or any aircraft transmitting (RX).
When the frequency goes quiet, your music fades smoothly back to normal.

Perfect for ATC, and anyone using TrackAudio who wants clear comms without manually adjusting audio.

Features
  Automatic Spotify volume ducking  
Reacts instantly to TrackAudio TX/RX events.

  Smooth fade in/out  
Adjustable fade‑down and fade‑up durations.

  Custom mute level  
Choose how low Spotify should drop during transmissions.

  Auto‑reconnect  
If TrackAudio restarts, the app reconnects automatically.

  Simple, modern UI  
Clean sliders, live value labels, and clear status indicators.

  Installation
Download the latest release from the Releases section.

Run ControllerMusic.exe.

No setup required — it just works.

  Usage
Start TrackAudio

Open Controller Music.

Adjust:

Mute Volume (%)

Fade Down (s)

Fade Up (s)

Click Start Monitoring.

The app will connect to TrackAudio and begin listening for transmissions.

Whenever:

You transmit (PTT) → Spotify volume fades down

Any aircraft transmits → Spotify volume fades down

Frequency is quiet → Spotify volume fades back up
